## Key Metrics Definitions

| Metric | Definition | Formula |
|--------|------------|---------|
| CPA (Cost per Application) | Cost to generate one job application | Spend / Applications |
| CPC (Cost per Click) | Cost for one ad click | Spend / Clicks |
| CTR (Click-Through Rate) | Percentage of impressions that result in clicks | (Clicks / Impressions) x 100 |
| Conversion Rate | Percentage of clicks that result in applications | (Applications / Clicks) x 100 |
| CPQA (Cost per Qualified Application) | Cost for applications meeting qualification criteria | Spend / Qualified Applications |
| CPH (Cost per Hire) | Total cost to make one hire | Total Spend / Hires |
