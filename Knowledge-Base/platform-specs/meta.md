# Meta (Facebook/Instagram) - Platform Specifications

## Special Ad Category: Employment

**CRITICAL**: All employment/recruitment ads on Meta platforms MUST be designated as Special Ad Category: Employment. This applies to Facebook, Instagram, Messenger, and Audience Network.

### Special Ad Category Restrictions

When running employment ads, the following targeting options are **NOT available**:

| Restricted Targeting | Reason |
|---------------------|--------|
| Age | Anti-discrimination |
| Gender | Anti-discrimination |
| Zip code (detailed) | Redlining prevention |
| Demographics | Anti-discrimination |
| Behaviors | Potential discrimination |
| Lookalike Audiences | Standard lookalikes disabled |

### Available Targeting (Employment Ads)

| Targeting Type | Availability |
|----------------|--------------|
| Location | Minimum 15-mile radius |
| Interests | Limited set only |
| Special Ad Audiences | Lookalike alternative |
| Custom Audiences | From own data sources |

## Ad Formats for Recruitment

### 1. Single Image Ads

| Element | Specification |
|---------|---------------|
| Image ratio | 1.91:1 to 1:1 (1:1 recommended) |
| Image size | 1080x1080 pixels minimum |
| File size | 30MB max |
| Primary text | 125 characters recommended (max varies) |
| Headline | 40 characters recommended (max 255) |
| Description | 30 characters recommended (link description) |

**Feed Placements**:
- Facebook Feed: 1:1 or 4:5
- Instagram Feed: 1:1 or 4:5
- Stories: 9:16

### 2. Video Ads

| Element | Specification |
|---------|---------------|
| Length | 1 second - 241 minutes (15-60 seconds recommended) |
| Ratio | 1:1 (feed), 9:16 (stories), 16:9 (in-stream) |
| Resolution | 1080p minimum |
| File size | 4GB max |
| Captions | Recommended (85% watch without sound) |

### 3. Carousel Ads

| Element | Specification |
|---------|---------------|
| Cards | 2-10 cards |
| Image size | 1080x1080 pixels |
| Video | Supported in carousel |
| Headline per card | 40 characters recommended |
| Description per card | 20 characters recommended |

Use case: Multiple roles, day-in-the-life, employee spotlights.

### 4. Collection Ads

| Element | Specification |
|---------|---------------|
| Cover image/video | Required |
| Product cards | 4+ items |
| Instant Experience | Required backend |

[VERIFY] Collection ad effectiveness for recruitment.

### 5. Lead Generation Ads

Native lead forms within Facebook/Instagram.

| Element | Specification |
|---------|---------------|
| Form fields | Up to 15 questions |
| Pre-fill options | Name, email, phone (from profile) |
| Custom questions | Multiple choice, short answer |
| Privacy policy | Required link |
| Thank you screen | Customizable |

**Advantage**: Lower friction than website forms.

## Ad Copy Best Practices

### Employment Ad Restrictions
- No age-related language ("young," "recent graduate")
- No gender-specific language
- No references to physical attributes
- No national origin preferences
- Focus on job requirements and benefits

### Effective Elements
- Employee testimonials
- Day-in-the-life content
- Company culture highlights
- Benefits and perks
- Growth opportunities
- Remote/flexible work options

## Compliance & Approval

### Ad Review Process
- All ads reviewed before going live
- Employment ads receive additional scrutiny
- Review time: typically 24 hours [VERIFY]

### Common Rejection Reasons
1. Missing Special Ad Category designation
2. Discriminatory language or imagery
3. Misleading job information
4. Low-quality landing page
5. Prohibited content
